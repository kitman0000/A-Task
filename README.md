# A-Task
A-Task 是一款面向个人的行程任务管理网站

## 项目功能
	基本账号功能
	行程任务管理
	每日任务查看

## 技术框架
	前端：Vue.js + JQuery + Bootstrap + Nginx
	后端：Spring boot + Mybatis + Redis + MySql
	
## 项目依赖
	请参加Maven的pom.xml
	
## 参数配置
	MySql数据库连接字符串
	
	Redis密码
	
	阿里云AK
	
	JWT密钥
